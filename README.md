## Installation

#### Prerequisites

- NodeJS
- Environment variable named `GH_TOKEN` containing a [GitHub PAT](https://github.com/settings/tokens)

```console
$ git clone https://github.com/imbreydan/cloney.git
$ cd cloney
$ npm link
```

Now you can run `cloney` from anywhere!
